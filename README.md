#  Social Media Project

A responsive social media interface developed during the HTML5 & CSS3 course by Curso em Vídeo.

This project simulates a smartphone displaying different social media pages using HTML and CSS, with a focus on responsive layouts, iframes, positioning, and user interaction.

##  Technologies

- HTML5
- CSS3

##  What I practiced

- Responsive Design
- CSS Positioning
- iFrames
- Flexbox
- Navigation between pages
- Mobile-first concepts

##  Purpose

This project was developed as part of my Web Development learning journey to strengthen my front-end fundamentals.
